# AsyncAPI docs plugin

Render AsyncAPI documentation in a project. This plugin uses an official <a href="https://github.com/asyncapi/asyncapi-react" target="_blank">React component</a> to generate documentation for AsyncAPI 2.x and 3.x descriptions.

The AsyncAPI docs plugin supports Apache Kafka protocol bindings.

When added to your project, this plugin automatically detects AsyncAPI description files and includes them in the project as pages.
To be detected, the description files must have a `.yaml` or `.json` extensions.

This is a plugin reference page. A setup guide can be found in the [How-to guide](../../../../author/how-to/generate-asyncapi-docs.md)
