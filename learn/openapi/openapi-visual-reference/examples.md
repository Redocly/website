# Named examples

You can name examples in the components object.

```yaml
components:
  examples:
    lightning:
      # Example object
```

It is a map with strings as the name, and the [Example Object](./example.md) as the value.

## Visuals

See the [Example Object](./example.md).

## Types

- NamedExamples
- Example

